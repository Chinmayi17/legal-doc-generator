# LegalEase - AI-Powered Legal Document Generator

Generate professional legal documents in seconds using Google Gemini AI.

## Features

- **Employment Contracts** - Complete employment agreements with standard clauses
- **Non-Disclosure Agreements (NDAs)** - Professional NDAs with confidentiality terms
- **Lease Agreements** - Residential lease contracts with all standard provisions
- **Service Agreements** - Professional service contracts with payment terms

## Tech Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **AI Model**: Google Gemini 2.5 Flash
- **Environment**: Python 3.12+

## Installation

1. Clone the repository:
```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
```

2. Create virtual environment:
```bash
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Set up environment variables:
Create a `.env` file in the project root:
```
GEMINI_API_KEY=your_gemini_api_key_here
```


## Usage

1. Start the Flask server:
```bash
python app.py
```

2. Open your browser and navigate to:
```
http://127.0.0.1:5000
```

3. Select document type, fill in the required fields, and click "Generate Document"

## API Documentation

### POST /generate

Generates a legal document based on provided parameters.

**Request Body:**
```json
{
  "document_type": "employment_contract",
  "employer_name": "ABC Corp",
  "employee_name": "John Doe",
  "job_title": "Software Engineer",
  "start_date": "01-03-2026",
  "salary": "$100,000",
  "work_location": "New York",
  "additional_terms": "Remote work allowed"
}
```

**Response:**
```json
{
  "success": true,
  "document": "Generated document text..."
}
```
